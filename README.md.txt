AUTOMAÇÃO WEB

É um projeto de automação para logar e cadastar um usuário.

Instalação:

Java 8 ou superior instalado
Navegador chrome
Selenium WebDriver
Cucumber
JUnit

Configurações e Utilização

Adicionar na Libraries JRE System Libray [javaSE-1-8]
Configura a propriedade do resource para UTF-8
Rodar a bateria de teste através da classe Runner


1 - gera as tabelas CLIENTE, USUARIO
2 - popula a tabela USUARIO
3 - efetua login
4 - efetua cadastro de cliente
5 - com algumas validações de campos e de máscaras (mascaras de campos numericos não barram caracteres)
6 - página com título dinâmico
7 - possui campos de telefone e celular não obrigatórios
